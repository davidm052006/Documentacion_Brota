# 1. Tabla de Épicas
---

| Código | Nombre de la Épica                          | Descripción breve                                                                           |
| ------ | ------------------------------------------- | ------------------------------------------------------------------------------------------- |
| EP-01  | Perfilamiento del estudiante                | Crear, actualizar y generar recomendaciones basadas en el perfil vocacional del estudiante. |
| EP-02  | Navegación y avance en rutas de aprendizaje | Permitir seguimiento, avance e incentivos dentro de las rutas de aprendizaje.               |
| EP-03  | Comparación y elección de carreras          | Facilitar la comparación clara entre dos carreras para apoyar la toma de decisiones.        |
| EP-04  | Interacción con la Mascota IA               | Proveer un asistente conversacional que responda preguntas sobre carreras y rutas.          |
| EP-05  | Gestión administrativa del contenido        | Administrar carreras, instituciones y contenidos del sistema.                               |
| EP-06  | Reportes para padres                        | Permitir que padres con permiso puedan ver el perfil y progreso del estudiante.             |
| EP-07  | Exportación de perfil y rutas               | Generar un PDF descargable con todo el perfil vocacional del usuario.                       |
