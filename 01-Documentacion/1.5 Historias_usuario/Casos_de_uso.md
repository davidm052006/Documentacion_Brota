
### 4.  Casos_de_uso.

| Categoría     | Función                                           | Valor añadido                                                             |
| ------------- | -------------------------------------------------- | ------------------------------------------------------------------------- |
| Evaluación    | Completar cuestionario por primera vez             | Obtiene perfil y recomendaciones totalmente personalizadas                |
| Evaluación    | Repetir cuestionario cada 3 meses                  | Actualiza su perfil para reflejar cambios reales                          |
| Decisión      | Comparar dos carreras                              | Ve una tabla clara y puede decidir con mejor información                  |
| Progreso      | Marcar contenido de ruta como completado           | Avanza en la barra de progreso y gana insignias                           |
| Asistencia    | Preguntar a la mascota IA                          | Recibe una respuesta honesta + ruta recomendada (ej. matemáticas)         |
| Acceso rápido | Usar cuestionario sin registro                     | Puede obtener perfil y recomendaciones sin crear cuenta                    |
| Padres        | Ver reporte del hijo/a (con permiso)               | Entiende la carrera recomendada y cómo apoyar                             |
| Administración| Añadir nueva carrera o institución                 | Se actualiza para todos en tiempo real                                    |
| Exportación   | Descargar PDF con perfil y rutas                   | Puede llevarlo a orientación escolar o a entrevistas                       |
| Engagement    | Recibir recordatorios diarios si no avanza         | Reduce el abandono y mantiene al usuario progresando                       |
