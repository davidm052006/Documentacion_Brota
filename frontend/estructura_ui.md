# 🎨 Estructura UI - Frontend Brota

## Componentes Principales

### 1. Cuestionario Vocacional

Componente interactivo para capturar respuestas del estudiante.

**Subcomponentes:**

- Pregunta individual
- Barra de progreso
- Navegación (anterior/siguiente)
- Validación de respuestas

### 2. Perfil Vocacional

Visualización del perfil generado con porcentajes por área.

**Subcomponentes:**

- Gráfico de habilidades
- Descripción de áreas fuertes
- Resumen del perfil

### 3. Recomendaciones

Lista de programas recomendados ordenados por compatibilidad.

**Subcomponentes:**

- Tarjeta de programa
- Porcentaje de compatibilidad
- Información de institución
- Botón de detalle

### 4. Detalle de Programa

Vista completa de un programa específico.

**Subcomponentes:**

- Información general
- Requisitos de ingreso
- Convocatoria activa
- Datos de contacto

### 5. Comparador de Programas

Vista lado a lado de múltiples programas.

### 6. Panel Administrativo

Interfaz para gestión de instituciones, programas y convocatorias.

**Subcomponentes:**

- Formularios de creación/edición
- Tablas de listado
- Filtros y búsqueda
