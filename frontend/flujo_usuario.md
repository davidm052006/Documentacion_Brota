# 🚶 Flujo de Usuario - Frontend Brota

## Flujo Principal: Estudiante

### 1. Inicio

Usuario llega a la página principal de Brota.

**Acciones:**

- Ver información sobre el sistema
- Botón "Comenzar cuestionario"

### 2. Cuestionario Vocacional

Usuario responde preguntas sobre habilidades, intereses y contexto.

**Acciones:**

- Responder preguntas
- Navegar entre preguntas (anterior/siguiente)
- Ver progreso
- Guardar y continuar después (opcional)

### 3. Generación de Perfil

Sistema procesa respuestas y genera perfil vocacional.

**Feedback:**

- Indicador de carga
- Mensaje de procesamiento

### 4. Visualización de Perfil

Usuario ve su perfil vocacional con porcentajes.

**Acciones:**

- Ver gráfico de habilidades
- Leer descripción de áreas fuertes
- Botón "Ver recomendaciones"

### 5. Recomendaciones

Usuario ve lista de programas recomendados.

**Acciones:**

- Ver tarjetas de programas
- Ordenar por compatibilidad
- Filtrar por tipo/modalidad
- Seleccionar programa para ver detalle
- Seleccionar programas para comparar

### 6. Detalle de Programa

Usuario ve información completa de un programa.

**Acciones:**

- Leer descripción completa
- Ver requisitos de ingreso
- Ver información de convocatoria
- Ver datos de contacto de institución
- Volver a recomendaciones

### 7. Comparación (Opcional)

Usuario compara hasta 3 programas lado a lado.

**Acciones:**

- Ver comparación de atributos
- Decidir entre opciones
- Volver a recomendaciones

### 8. Descarga de Reporte (Opcional)

Usuario descarga PDF con resultados.

**Acciones:**

- Botón "Descargar reporte"
- Recibir PDF con perfil y recomendaciones

## Flujo Secundario: Administrador

### 1. Login

Administrador ingresa credenciales.

### 2. Dashboard

Vista general de estadísticas.

### 3. Gestión

Administrador gestiona instituciones, programas y convocatorias.

**Acciones:**

- Crear/editar/eliminar entidades
- Buscar y filtrar
- Ver estadísticas
